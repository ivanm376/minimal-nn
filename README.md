# Simplified neural network

\- no external libraries (mostly scratched from **brain.js**)  
 \- single neuron (in example - see **const layers**)

# Description

Learns to turn any **[0-0.5, 0.5-1]** input array to **[0, 1]** and **[0.5-1, 0-0.5]** to **[1, 0]**  
Layers configuration: [2, 1, 2]
